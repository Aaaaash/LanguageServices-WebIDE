# LanguageServer

## run development

```bash
$ cp -r xxx/eclipse.jdt.ls/server xxx/LanguageServices-WebIDE/lsp-java-server
$ yarn install
$ yarn dev
```

## run production

```bash
$ cp -r xxx/eclipse.jdt.ls/server xxx/LanguageServices-WebIDE/lsp-java-server
$ yarn install
$ yarn build
$ yarn pm2
```
