# LanguageServer

[![Build Status](https://travis-ci.org/Aaaaash/LanguageServices-WebIDE.svg?branch=master)](https://travis-ci.org/Aaaaash/LanguageServices-WebIDE)

## run development

```bash
$ yarn install
$ yarn prepare:env
$ yarn dev
```

## run production

```bash
$ yarn install
$ yarn prepare:env
$ yarn build
$ yarn pm2
```
